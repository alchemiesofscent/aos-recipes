# Archived governance docs

These documents described an upstream LLM-candidate / human-review campaign for promoting all 193 recipes to the gold-standard schema. That workflow was superseded in 2026-06 when scholarly revision moved into the Claude cowork workflow, where each recipe is revised one at a time (text, translation, commentary, TEI links) outside this repository.

The docs are kept here as a historical record of the v1 design considerations. **For forward-looking schema recommendations distilled from this thinking, see [`../schema_recommendations.md`](../schema_recommendations.md).**

## Files

- [`schema_policy.md`](schema_policy.md) — formal data-model revision policy v0.1.0; F1–F19 gap analysis; W1–W5 workstreams.
- [`normalization_campaign.md`](normalization_campaign.md) — active campaign plan v0.1.0; LLM-candidate review contract and gating commands.

Both documents were authored 2026-06-05 and remain accurate as a description of the design that was considered. The substantive observations (gap analysis, design strengths to preserve, recommended workstreams) are extracted and reframed in `docs/schema_recommendations.md`.
