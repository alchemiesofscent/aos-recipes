# Documentation Index

This directory holds the documentation for `aos-recipes`. The entries below are grouped by audience. Each entry is one bullet: `[file](file) — one-line purpose`.

## Scope

`aos-recipes` is a corpus-first export of **193 QC-complete recipes** drawn from five slices: Dioscorides Book 1 perfumes & resins (52), Dioscorides Book 2 fats (18), Aëtius Book 1 oils (43), Aëtius Book 16 myrepsika (45), Paul 7.20 perfumes (35). The repository contains the canonical JSON corpus, locally derived enrichment (ingredient lexicon, co-occurrence analysis), a mirrored quantity-gold authority layer, and a no-build static browser under `site/`. It is **not** a source-extraction pipeline — recipe entities are produced upstream in `/home/seanm/github/aetius` and mirrored here for publication. The Aëtius Book 1 pre-emended backup and the comparison viewers in `provenance/` are retained for audit only; they are not peer recipe collections.

## Validation

After any regeneration, every script below should exit 0:

```bash
python3 scripts/validate.py
python3 scripts/extract_ingredient_lexicon.py --check
python3 scripts/cooccurrence.py --check
python3 scripts/quantity_gold.py --validate
```

`scripts/validate.py` checks schema shape, unique recipe IDs, per-slice counts, `derived_recipe_id` links, entity-file coverage, forbidden exported paths, quantity-gold overlay parity, and the co-occurrence index recipe count. `--require-complete` mode of `quantity_gold.py` is gated on the upstream review campaign and currently fails because only 1/193 recipes are reviewed; see [`release_v1.md`](release_v1.md).
